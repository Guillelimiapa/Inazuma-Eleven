Sobre esta web

Esta demo está hecha con HTML + CSS + JavaScript y usa un archivo JSON local como fuente de datos, por lo que es compatible con GitHub Pages sin necesidad de PHP ni base de datos.

Puedes ampliar fácilmente el archivo data/characters.json para añadir muchos más personajes, técnicas, equipos o imágenes personalizadas.
